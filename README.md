## GitHub Pages Deployment

This site is automatically deployed to GitHub Pages whenever you push changes to the main/master branch.

### Setup Instructions:

1. **Push to GitHub**: Ensure your repository is pushed to GitHub
2. **Enable GitHub Pages**:
   - Go to your repository Settings
   - Navigate to **Pages** section (left sidebar)
   - Select **Deploy from a branch**
   - Choose the branch: `gh-pages` (created automatically by the workflow)
   - Click **Save**

3. **View Your Site**: Your site will be live at `https://<your-username>.github.io/<repository-name>`

### Workflow

The `.github/workflows/deploy.yml` file automatically:
- Triggers on every push to main/master
- Deploys your site to the `gh-pages` branch
- Makes your site live at your GitHub Pages URL
